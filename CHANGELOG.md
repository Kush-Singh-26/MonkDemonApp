# Changelog

All notable changes to this project are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — Unreleased

### Added
- Standalone HTML puzzle converted into a Capacitor Android app
- Offline font support (Cinzel, IM Fell English, Courier Prime)
- Three built-in solvers: BFS, DFS, and A*
- Animated search-tree visualization with labeled nodes and edge badges
- Undo / Redo move history
- Auto-play solution at configurable speed
- Immersive fullscreen and portrait-lock on Android
- `.gitignore`, `README.md`, `CONTRIBUTING.md`, `CHANGELOG.md`, `BUILD.md`
- `PLAYSTORE.md` and `ASSETS_FOR_PLAYSTORE.md` checklists
- Debug APK and release AAB pre-built in `releases/`

### Known Issues
- Search tree nodes may appear blurry on some low-DPI emulators
- Placeholder icons and splash screens — replace before publishing
